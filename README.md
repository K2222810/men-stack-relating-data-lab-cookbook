# Men Stack Relating Data Lab Cookbook
A full-stack cookbook app built to practice relating data in MongoDB.

## Description
This project was built to practice working with related data in a full-stack app. It focuses on how different pieces of information connect together inside MongoDB, while also using Express and EJS to let users interact with the data.

It was a useful way to improve my understanding of database relationships, full-stack structure, and building more connected features.

## Planning
Before building, the main focus was understanding how the different data models would connect.

Planning included:

- Deciding what data needed relationships
- Planning model structure
- Thinking about how related data would be shown in views
- Organising routes and controllers clearly

## Brief
The goal of this lab was to practice relating data in a MEN stack application.

The app needed to show how linked data can be created, stored, and displayed in a full-stack project.

## Build Process
The project started with setting up the Express app and database connection. After that, the models were built so the related data could be stored and connected correctly.

Once the structure was in place, routes, controllers, and views were added so users could create and interact with the linked content.
